# Hanzo JavaScript SDK - TypeScript Client Library

**Category**: Hanzo Ecosystem
**Related Skills**: `hanzo/hanzo-chat.md`, `hanzo/python-sdk.md`, `hanzo/go-sdk.md`

## Overview

Hanzo JS SDK is a **TypeScript-first client library** for the Hanzo AI API. Generated by Stainless for type safety and API completeness. Works in Node.js and browsers.

### Why Hanzo JS SDK?

- **Full type safety**: Generated from OpenAPI spec
- **OpenAI compatible**: Same interface patterns
- **Browser + Node**: Universal JavaScript support
- **Streaming**: First-class SSE streaming support
- **Tree-shakable**: Only import what you need

## When to use

- TypeScript/JavaScript applications calling Hanzo API
- Next.js, React, or Node.js projects
- Browser-based AI applications
- Replacing OpenAI SDK with Hanzo

## One-file quickstart

```typescript
import Hanzo from "@hanzoai/sdk"

const client = new Hanzo({
  apiKey: process.env.HANZO_API_KEY,
})

// Chat completion
const response = await client.chat.completions.create({
  model: "zen-70b",
  messages: [{ role: "user", content: "Hello, Hanzo!" }],
})
console.log(response.choices[0].message.content)

// Streaming
const stream = await client.chat.completions.create({
  model: "zen-70b",
  messages: [{ role: "user", content: "Write a poem" }],
  stream: true,
})
for await (const chunk of stream) {
  process.stdout.write(chunk.choices[0]?.delta?.content || "")
}

// Embeddings
const embedding = await client.embeddings.create({
  model: "zen-embedding",
  input: "Hello world",
})
```

### With OpenAI SDK (drop-in)

```typescript
import OpenAI from "openai"

const client = new OpenAI({
  baseURL: "https://api.hanzo.ai/v1",
  apiKey: process.env.HANZO_API_KEY,
})
// Everything works the same
```
