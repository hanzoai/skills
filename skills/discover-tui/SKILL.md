# Tui Skills Discovery

Provides automatic access to comprehensive tui skills.

## When This Skill Activates

This skill auto-activates when you're working with:
- terminal user interfaces
- TUI
- Bubble Tea
- Ratatui
- terminal UI
- text-based interfaces
- ncurses
